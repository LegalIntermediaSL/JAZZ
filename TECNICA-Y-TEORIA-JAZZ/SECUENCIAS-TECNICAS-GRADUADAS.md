# Secuencias tecnicas graduadas

Este documento organiza la parte tecnica por niveles de dificultad. No todas las personas necesitan entrar por el mismo sitio, y no toda escucha tecnica equivale ya a improvisar o analizar armonia con detalle.

## Como usar estas secuencias

- elige el nivel por tu oido real, no por orgullo
- no avances solo por haber terminado un texto
- vuelve atras si la escucha pierde claridad
- cruza siempre teoria con piezas concretas

## 6. Cuatro secuencias ultrabreves

### Para reconocer forma

- blues
- AABA
- tema y solos

### Para reconocer armonia funcional

- Autumn Leaves
- All the Things You Are

### Para reconocer modal

- So What
- Maiden Voyage

### Para reconocer diferencia entre version y obra

- un standard vocal
- una version instrumental del mismo

## 7. Preguntas utiles al subir de nivel

- oyes mejor la forma sin contar de manera mecanica
- distingues mejor una version de otra
- relacionas el sonido con decisiones concretas
- puedes nombrar sin romper la escucha
