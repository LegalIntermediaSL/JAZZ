# JAZZ

Repositorio documental en Markdown dedicado a la musica jazz. Reune una guia de iniciacion, un registro de trabajo del proyecto y un historial de cambios para construir una base clara, ampliable y util para estudio, divulgacion o clases.

## Vision del proyecto

JAZZ busca ofrecer una entrada accesible al genero sin perder profundidad. La idea no es solo listar nombres famosos, sino ordenar el aprendizaje alrededor de tres ejes:

- contexto historico
- estilos principales
- escucha guiada de artistas, canciones y albumes

## Para quien es esta guia

Esta guia esta pensada para personas que quieren entrar en el jazz sin sentirse perdidas. No hace falta saber teoria musical: basta con curiosidad y ganas de escuchar.

## Como usar este repositorio

Si es tu primera vez con el jazz, esta es una ruta simple:

1. Lee esta portada para entender el enfoque del proyecto.
2. Continua con [GUIA-JAZZ.md](./GUIA-JAZZ.md) para recorrer estilos, artistas y discos.
3. Pasa a [CRONOLOGIA-JAZZ.md](./CRONOLOGIA-JAZZ.md) para ubicar cada estilo en el tiempo.
4. Entra en [HISTORIA-DEL-JAZZ/](./HISTORIA-DEL-JAZZ/README.md) si quieres una lectura historica mas narrativa.
5. Usa [PLAYLISTS-JAZZ.md](./PLAYLISTS-JAZZ.md) para convertir la lectura en escucha guiada.
6. Entra en [ESTILOS/](./ESTILOS/README.md) si quieres estudiar el jazz por lenguajes concretos.
7. Entra en [INTERPRETES/](./INTERPRETES/README.md) si prefieres aprender a partir de artistas clave.
8. Entra en [APRENDER-JAZZ-OYENDO-ALBUMES/](./APRENDER-JAZZ-OYENDO-ALBUMES/README.md) si quieres formar el oido a traves de discos completos.
9. Consulta [BITACORA.md](./BITACORA.md) si quieres ver el criterio editorial y el rumbo del contenido.
10. Revisa [CHANGELOG.md](./CHANGELOG.md) para conocer que se ha agregado o actualizado.

## Que incluye la guia principal

La guia actual ofrece una base de trabajo suficientemente solida para empezar:

- definicion sencilla de que es el jazz
- conceptos basicos como swing, standard, improvisacion y combo
- recorrido historico desde New Orleans hasta el jazz contemporaneo
- artistas imprescindibles por instrumento y rol
- lista inicial de canciones esenciales
- seleccion de albumes fundamentales
- ruta de escucha para principiantes
- cronologia comentada por periodos
- una carpeta especifica para historia del jazz
- playlists tematicas para profundizar por estilo, clima o nivel
- carpetas especializadas para estilos, interpretes y aprendizaje por albumes

## Estado actual

El proyecto ya cuenta con una base documental inicial compuesta por guia, cronologia, playlists y cuatro carpetas tematicas de desarrollo. Aun asi, sigue en crecimiento y todavia hay margen para refinar el material con perfiles mas amplios, glosarios, recursos didacticos y recorridos especializados.

## Hoja de ruta sugerida

- desarrollar secciones dedicadas a vocal jazz, latin jazz y fusion
- incorporar perfiles breves de interpretes esenciales
- sumar recomendaciones para docentes o autodidactas
- crear glosarios, fichas de albumes y recorridos por instrumentos
- enlazar mejor los documentos entre carpetas y crear indices cruzados
- ampliar la carpeta de historia con hitos sociales, geografias y escenas locales

## Criterio editorial

El contenido del proyecto prioriza claridad, progresion y capacidad de escucha. La seleccion no pretende ser definitiva ni enciclopedica; funciona como una puerta de entrada curada para descubrir el genero paso a paso.
