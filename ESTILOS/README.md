# Estilos de jazz

## Proposito

Esta seccion organiza el jazz por estilos y periodos. La idea es que cada documento te ayude a reconocer como cambia el lenguaje del genero segun la epoca, el contexto y la forma de tocar.

## Como recorrer esta carpeta

1. Empieza por los origenes y el swing para entender el pulso historico.
2. Sigue con bebop, cool y hard bop para entrar en el jazz moderno.
3. Termina con modal, free, fusion y escena contemporanea para abrir el oido a otras formas.

- distinguir estilos por ritmo, sonido y forma
- reconocer artistas que definen una epoca
- relacionar canciones y albumes con un contexto historico
