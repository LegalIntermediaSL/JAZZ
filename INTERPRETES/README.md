# Interpretes de jazz

## Proposito

Esta seccion organiza el proyecto alrededor de musicos y musicas esenciales. En lugar de estudiar el jazz solo por estilos, aqui puedes seguirlo por voces personales.

## Como recorrer esta carpeta

1. Empieza por trompeta y saxo para escuchar el frente melodico del jazz.
2. Sigue con piano y seccion ritmica para entender como se sostiene la musica.
3. Termina con las voces esenciales para entrar en el fraseo vocal y la interpretacion de standards.
