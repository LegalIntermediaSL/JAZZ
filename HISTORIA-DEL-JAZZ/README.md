# Historia del jazz

## Proposito

Esta carpeta reune materiales para estudiar la historia del jazz como proceso cultural, musical y social. A diferencia de una cronologia rapida, aqui el objetivo es recorrer etapas con algo mas de contexto y continuidad narrativa.

## Como recorrer esta carpeta

1. Empieza por los origenes para entender de donde sale el lenguaje del jazz.
2. Sigue con swing y bebop para ver el paso del baile a la modernidad.
3. Termina con la expansion global del jazz para comprender como el genero se multiplica y se transforma.
