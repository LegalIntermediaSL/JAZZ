# Galerias tematicas de dominio publico

Este documento organiza posibles ampliaciones visuales por temas. No incluye imagenes insertadas en masa. Ofrece criterios y agrupaciones para buscar, seleccionar y reutilizar materiales con mas sentido.

## 4. Por soportes visuales

- retrato promocional
- foto de escenario
- portada historica
- programa o cartel

## 5. Criterios para elegir bien

- fuente clara
- nota explicita de dominio publico o equivalencia compatible
- identificacion minima de autor, fecha y contexto cuando exista
- relacion real con el contenido del proyecto

## 6. Donde buscar

- Wikimedia Commons
- Library of Congress
- archivos publicos institucionales
- colecciones digitales de museos y bibliotecas
