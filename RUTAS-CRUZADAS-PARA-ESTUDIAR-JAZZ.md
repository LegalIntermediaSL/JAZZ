# Rutas cruzadas para estudiar jazz

Este documento intenta resolver un problema muy habitual: conocer ya varios nombres, discos o estilos, pero no tener claro como pasar de una parte del proyecto a otra. No propone una sola ruta obligatoria. Propone puentes.

Para una orientacion general, usa tambien [README.md](./README.md), [GUIA-JAZZ.md](./GUIA-JAZZ.md) y [GLOSARIO-DE-TERMINOS-JAZZ.md](./GLOSARIO-DE-TERMINOS-JAZZ.md).

## Como usar este indice

- elige primero una pregunta real, no una ambicion demasiado grande
- combina siempre un documento de contexto con uno de escucha
- suma despues una capa de artista, album o tecnica
- vuelve aqui cuando sientas que sabes nombres, pero aun no sabes relacionarlos

## 1. Si tu pregunta es historica

Esta es la mejor ruta cuando quieres entender de donde sale cada sonido.

1. Empieza por [CRONOLOGIA-JAZZ.md](./CRONOLOGIA-JAZZ.md).
2. Sigue con [HISTORIA-DEL-JAZZ/README.md](./HISTORIA-DEL-JAZZ/README.md).
3. Baja luego a [HISTORIA-DEL-JAZZ/CIUDADES-Y-ESCENAS-CLAVE.md](./HISTORIA-DEL-JAZZ/CIUDADES-Y-ESCENAS-CLAVE.md) y [HISTORIA-DEL-JAZZ/INDUSTRIA-MEDIOS-Y-FESTIVALES.md](./HISTORIA-DEL-JAZZ/INDUSTRIA-MEDIOS-Y-FESTIVALES.md).
4. Cruza despues con [ESTILOS/README.md](./ESTILOS/README.md) para oir mejor cada etapa.
5. Cierra con [APRENDER-JAZZ-OYENDO-ALBUMES/RUTA-DE-10-ALBUMES.md](./APRENDER-JAZZ-OYENDO-ALBUMES/RUTA-DE-10-ALBUMES.md) o [PLAYLISTS-JAZZ.md](./PLAYLISTS-JAZZ.md).

### Lo que deberias poder responder

- de donde viene el jazz
- por que cambian los formatos y los lenguajes
- como se relacionan ciudades, escenas, industria y sonido

## 3. Si tu puerta de entrada es un artista

Esta es la ruta mas util cuando ya tienes un nombre favorito y no quieres quedarte solo ahi.

1. Empieza por [INTERPRETES/README.md](./INTERPRETES/README.md).
2. Usa [INTERPRETES/PERFILES-CORTOS-DE-ARTISTAS-ESENCIALES.md](./INTERPRETES/PERFILES-CORTOS-DE-ARTISTAS-ESENCIALES.md) si aun no sabes bien con quien empezar.
3. Usa [INTERPRETES/DISCOGRAFIAS-GUIADAS.md](./INTERPRETES/DISCOGRAFIAS-GUIADAS.md) para elegir discos.
4. Pasa luego a [INTERPRETES/RUTAS-POR-FAMILIAS-E-INFLUENCIAS.md](./INTERPRETES/RUTAS-POR-FAMILIAS-E-INFLUENCIAS.md) para situar esa voz dentro de una linea mas amplia.

6. Vuelve por ultimo a la carpeta de [ESTILOS/](./ESTILOS/README.md) o a [HISTORIA-DEL-JAZZ/](./HISTORIA-DEL-JAZZ/README.md) para poner ese recorrido en contexto.

Esta via sirve cuando recuerdas un disco concreto y quieres construir alrededor suyo.

1. Entra por [APRENDER-JAZZ-OYENDO-ALBUMES/README.md](./APRENDER-JAZZ-OYENDO-ALBUMES/README.md).
2. Usa [APRENDER-JAZZ-OYENDO-ALBUMES/PRIMERAS-ESCUCHAS-POR-NIVEL.md](./APRENDER-JAZZ-OYENDO-ALBUMES/PRIMERAS-ESCUCHAS-POR-NIVEL.md) si necesitas calibrar dificultad.

4. Sigue con [INTERPRETES/DISCOGRAFIAS-GUIADAS.md](./INTERPRETES/DISCOGRAFIAS-GUIADAS.md) para continuar la trayectoria del artista.

## 10. Formula minima para no perderte

Si no sabes como montar una ruta propia, usa esta combinacion:

1. un documento de marco historico o estilistico
2. un documento de escucha concreta
3. un documento de artistas o discografias
4. un documento tecnico o de comparacion
5. un documento cultural o visual

### Ejemplo simple

- marco: [ESTILOS/BEBOP-COOL-Y-HARD-BOP.md](./ESTILOS/BEBOP-COOL-Y-HARD-BOP.md)
- escucha: [APRENDER-JAZZ-OYENDO-ALBUMES/RUTA-DE-10-ALBUMES.md](./APRENDER-JAZZ-OYENDO-ALBUMES/RUTA-DE-10-ALBUMES.md)
- artistas: [INTERPRETES/DISCOGRAFIAS-GUIADAS.md](./INTERPRETES/DISCOGRAFIAS-GUIADAS.md)
- tecnica: [TECNICA-Y-TEORIA-JAZZ/FICHAS-DE-STANDARDS-CLAVE.md](./TECNICA-Y-TEORIA-JAZZ/FICHAS-DE-STANDARDS-CLAVE.md)
- cultura: [CULTURA-JAZZ/LUGARES-CLUBES-Y-CIUDADES.md](./CULTURA-JAZZ/LUGARES-CLUBES-Y-CIUDADES.md)

## Idea final

Estudiar jazz no consiste en elegir entre historia, estilos, artistas o albumes. Consiste en aprender a saltar bien entre ellos. Cuando esos saltos empiezan a salir solos, el oido ya esta creciendo de verdad.
