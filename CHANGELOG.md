# Changelog

Todos los cambios importantes de este proyecto se registran en este archivo.

El formato toma como referencia Keep a Changelog y utiliza fechas en estilo ISO `YYYY-MM-DD`.

## [0.6.0] - 2026-06-13

### Changed

- reescritura amplia de `README.md` y `GUIA-JAZZ.md` para darles mas profundidad pedagogica y mejor navegacion
- ampliacion fuerte de `CRONOLOGIA-JAZZ.md` y de la carpeta `HISTORIA-DEL-JAZZ/`
- desarrollo mucho mas detallado de las carpetas `ESTILOS/`, `INTERPRETES/` y `APRENDER-JAZZ-OYENDO-ALBUMES/`
- expansion de `PLAYLISTS-JAZZ.md` con objetivos de escucha, discos puente y rutas de uso

## [0.5.0] - 2026-06-13

### Added

- carpeta `HISTORIA-DEL-JAZZ/` con una seccion inicial dedicada al desarrollo historico del genero
- documentos base sobre origenes, transicion del swing al bebop y expansion moderna y global

### Changed

- ampliacion de `README.md` para integrar la nueva carpeta historica en la navegacion principal
- actualizacion de `BITACORA.md` con el nuevo alcance de la arquitectura editorial

## [0.4.0] - 2026-06-13

### Added

- carpeta `ESTILOS/` con documentos base para estudiar la evolucion del jazz por lenguajes
- carpeta `INTERPRETES/` con perfiles iniciales de trompeta, saxo, piano, seccion ritmica y voces
- carpeta `APRENDER-JAZZ-OYENDO-ALBUMES/` con una ruta de discos y una guia de escucha

### Changed

- ampliacion de `README.md` para reflejar la nueva estructura por carpetas
- actualizacion de `BITACORA.md` con el nuevo alcance tematico del proyecto

## [0.3.0] - 2026-06-13

### Added

- archivo `CRONOLOGIA-JAZZ.md` con una linea historica comentada del jazz
- archivo `PLAYLISTS-JAZZ.md` con rutas de escucha por estilos, climas y niveles
- nuevas referencias cruzadas entre la portada y los documentos de contenido

### Changed

- ampliacion de `README.md` para incorporar la nueva arquitectura editorial del proyecto
- actualizacion de `BITACORA.md` con el nuevo alcance y los siguientes hitos de contenido

## [0.2.0] - 2026-06-13

### Added

- archivo `BITACORA.md` para registrar decisiones editoriales, avances y siguientes pasos
- archivo `CHANGELOG.md` para documentar la evolucion del repositorio
- nuevos enlaces internos desde la portada hacia la documentacion principal

### Changed

- ampliacion de `README.md` con vision del proyecto, estructura, estado actual y hoja de ruta
- reorganizacion del repositorio como una base documental en Markdown

## [0.1.0] - 2026-06-13

### Changed
